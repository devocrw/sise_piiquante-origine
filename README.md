sise_piiquante-origine

Dans le dossier backend/config renommer le fichier .env.reference en .env, puis renseigner vos accès après les "=" :

DB_USER_PASS= ...?

TOKEN_SECRET= ...?

COOKIESECRET= ...?

EMAIL_KEY_CRYPTO= ...?

DB_URL= ...?

TECHNOLOGIES UTILISÉES

● framework : Express

● serveur : NodeJS

● base de données : MongoDB Atlas
